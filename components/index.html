Components

NEUMO Components are interactive building blocks for creating a user interface.

Buttons

Buttons help people initiate actions, from sending an email, to sharing a document, to liking a post.
There are three types of common buttons: gradient, filled, plain, and text.

Toggle Switch

Switches are very convenient for you to handle some events like on-off transitions.

Select

Text Input

The Neumo input CSS class make use of input tag of HTML. It will help you to apply neumorphism to your input forms.
